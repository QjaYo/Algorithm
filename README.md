# Algorithm

알고리즘및실습 수업 실습자료를 바탕으로 함.

C언어로 구현

### 목차정리

2주차_우선순위큐
- 선택정렬
- 삽입정렬

3주차_힙과힙정렬
- 삽입식 힙생성
- 상향식 힙생성

4주차_힙과힙정렬
- 힙정렬

5주차_합병정렬과퀵정렬
- 합병정렬
- 퀵정렬

6주차_사전
- 사전

7주차_탐색트리
- 이진탐색트리

8주차_해시테이블
- 분리연쇄법
- 개방주소법(선형,이중해싱)

9주차_중간고사

10주차_그래프
- 인접리스트
- 인접행렬

11주차_그래프순회
- DFS(깊이우선탐색)
- BFS(너비우선탐색)

12주차_방향그래프
- Floyd-Warshall 알고리즘 (Transitive Closure)
- Topological Order 구하기 (inDegree활용 / DFS활용)

13주차_최소신장트리
- Prim-Jarnik 알고리즘
- Krustal 알고리즘
- Baruvka 알고리즘

14주차_최단경로
- Dijkstra 알고리즘
- Bellman-Ford 알고리즘
- DAG 최단경로 알고리즘
- 모든 쌍 최단경로
